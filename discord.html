Discord-API-Limits

This python library is an API wrapper for Discord and is available to download from PyPi.

This lib allows you to easily make calls to the Discord API without having to worry about rate limits. It breaks down all the paths in the Discord API allowing you to quickly make calls just with the simple task of the specific function. All the API paths have their own function that supports nearly all the parameters that every path accepts. Images and files are not yet supported, I unfortunately do not currently have the proper understanding of how to share files with an API call.
