Cloudflare_DNS_updater

This is a simple python script that you can set to run on a schedule in the background and it will update DNS records if you are self-hosting applications and don’t have access to a static IP for your home.

Using the Cloudflare API this script will check to see if your networks public IP address has changed and if it has it simply updates Cloudflare’s DNS service to point at the new IP. This is useful for people that self-host applications that they want to have access to outside of their local network. This removes the need for a static IP as you can rest assured that whenever your public IP gets changed your services will not be affected.
